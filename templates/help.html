Program Features
 

Sentimental Analysis

Using Key Words, we analyse the emotions of the posts and whether if the posts require immediate attention.

Machine Learning

Trained and tested data based on NLP in order to quickly and precisely pinpoint locations that have been negatively influenced by natural disasters.

Automatic Solution

Based on historic and training datasets, an automatic solution would be suggested for when certain natural disasters strike.

Image Classification

Detect whether if the image posted is related to a natural disaster.
